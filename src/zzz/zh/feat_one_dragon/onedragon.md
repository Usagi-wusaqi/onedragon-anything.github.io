## 一条龙运行

一条龙主要集成大量日常功能，按需调整对应功能可使其按一定顺序进行执行。

> 若对应功能无括号说明，可点击功能名跳转到对应详细说明

|  功能   | 说明  |
|  ----  | ----  |
| 兑换码⚙️ | 每次自动检测并兑换全新的 `兑换码`（可手动配置或依赖前瞻兑换码自动更新） |
| 邮件 | 每天自动打开 `邮件` 尝试全部领取（无论邮件里是否有内容） |
| [体力刷本⚙️](./charge_plan.md) | 每次自动打开 `快捷手册-训练` 按配置尝试完成体力计划 |
| [恶名狩猎⚙️](./notorious_hunt.md) | 每周自动打开 `快捷手册-训练` 按配置完成恶名狩猎周期挑战 |
| [随便观⚙️](./suibian.md) | 每天自动前往 `澄辉坪-随便观` 按配置完成相关任务 |
| 刮刮卡 | 每天自动前往 `六分街-报刊亭` 完成刮刮卡（可以不用与卦象集录同时开启） |
| 卦象集录 | 每天自动前往 `澄辉坪-阿朔` 完成卦象集录（可以不用与刮刮卡同时开启） |
| [录像店营业⚙️](./random_play.md) | 每天自动按配置传送到指定地点完成录像店营业 |
| [咖啡店⚙️](./coffee.md) | 每天自动按配置传送到指定地点完成喝咖啡等任务 |
| 活跃度奖励 | 每天自动打开 `快捷手册-日常` 领取活跃度奖励（请尽量将该任务移到偏后的位置） |
| 丽都城募 | 每天自动打开 `丽都城募` 领取经验和奖励（已兼容开启丽都城募的一次性提示） |

| 驱动盘拆解⚙️ | 每天自动打开 `驱动仓库` 按配置完成驱动盘拆解（默认仅拆解 `A及以下`） |
| [枯萎之都⚙️](./hollow_zero.md#2零号空洞-枯萎之都) | 每周自动打开 `快捷手册-作战` 按配置完成枯萎之都的相关挑战 |
| [迷失之地⚙️](./hollow_zero.md#1零号空洞-迷失之地) | 每周自动打开 `快捷手册-作战` 按配置完成迷失之地的相关挑战 |
| [式舆防卫战⚙️](./shiyu_defense.md) | 每天自动打开 `快捷手册-作战` 按配置完成式舆防卫战相关挑战 |
| [锄大地⚙️](./world_patrol.md) | 每天自动前往 `航天城` 按配置清理野怪以刷取基础材料 |
| [真·拿命验收⚙️](./life_on_line.md) | 每天自动前往 `H.D.D.` `第二章间章-战斗委托-真·拿命验收` 按配置刷取丁尼 |

| 通知 | 每天自动发送一条龙运行情况的总览通知 |

## 预备编队

1. 编队名称
    - 请确保编队名称与游戏内完全一致，顺序随意，名称不匹配会导致无法识别
    - 不建议使用默认的数字命名编队，OCR 识别数字容易出错，建议改用中文名称
    > 除非你只使用 `游戏内编队` 这个选项

2. 自动识别
    - 点击「开始」后将自动打开游戏内预备编队页面，通过截图识别各编队中的代理人并填入左侧配置
    > 编队名称尽量<font color="red">不要只有一个字或者只有数字</font>的差异，以防识别错误

## 灵敏度校准

自动校准鼠标/手柄的转向灵敏度，用于视角转动
