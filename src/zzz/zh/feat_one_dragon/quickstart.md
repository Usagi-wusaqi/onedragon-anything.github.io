使用本页说明的功能时，建议阅读以下内容：
::: important
- 不同标签页功能不可同时启用
- 各功能使用之前需要去对应的功能页进行相关设置
:::

一条龙整合大量日常功能，按需开启对应功能，使其按一定顺序进行执行。

<font color="red">游戏内所有的点一次就会消失的不再提醒，都需要自己先点一次，脚本不会做兼容，因为几乎无法测试。</font>

> 点击名字可跳转对应功能的详细说明，带⚙️的为一条龙运行可配置项
>
> 文档在一条龙运行的排列顺序上做了微调，以符合常规阅读或使用习惯

|  功能   | 说明  |
|  ----  | ----  |
| 兑换码⚙️ | 每次自动兑换检测到的前瞻或手动添加的兑换码 |
| 邮件 | 每天自动尝试全部领取邮件内的奖励 |
| 录像店营业⚙️ | 每天自动进行一次录像店的营业上架，默认 `随机`。可选录像店代理人 |
| 刮刮卡 | 每天自动完成报刊亭的刮刮卡（可以不用与 `卦象集录` 同时开启） |
| 卦象集录 | 每天自动完成卦象集录的占卜（可以不用与 `刮刮卡` 同时开启） |
| [随便观⚙️](./suibian.md) | 每天自动完成随便观的相关任务 |
| [恶名狩猎⚙️](./notorious_hunt.md) | 每周自动完成恶名狩猎计划 |
| [体力刷本⚙️](./charge_plan.md) | 每次自动依次完成体力计划 |
| [咖啡店⚙️](./coffee.md) | 每天自动前往咖啡店执行喝咖啡任务 |
| 活跃度奖励 | 每天自动领取 `快捷手册-日常` 的每日活跃度奖励 |
| 丽都城募 | 每天自动领取 `丽都城募` 的奖励（兼容新版本可能造成卡住的首次弹窗） |
| 丽都周纪（领奖励） | 每天自动领取 `丽都周纪` 的活跃度奖励，每周拿下至少 800 活跃度可拿满菲林奖励。~~剩下那点是蚊子腿~~ |
| 驱动盘拆解⚙️ | 每天自动拆解驱动盘，默认只拆解 `A及以下`。可选拆解等级（`B`，`A及以下`，`S及以下`）和 `全部已弃置` |
| [枯萎之都⚙️](./hollow_zero.md#1零号空洞-枯萎之都) | 每周自动完成 `零号空洞-枯萎之都` 的相关任务 |
| [迷失之地⚙️](./hollow_zero.md#2零号空洞-迷失之地) | 每周自动完成 `零号空洞-迷失之地` 的相关任务 |
| [锄大地⚙️](./world_patrol.md) | 每天自动完成锄大地的相关任务 |
| [真·拿命验收⚙️](../feat_game_assistant.md#拿命验收) | 每天自动完成 `真·拿命验收` 副本，配置项位于 `游戏助手` |
| [式舆防卫战⚙️](./shiyu_defense.md) | 每天自动完成式舆防卫战的相关任务 |
| 通知⚙️ | 每天自动发送一条龙运行情况的总览通知 |
| [预备编队⚙️](../feat_game_assistant.md#预备编队) | 设置仅用于避免选择配队冲突，`预备编队识别` 位于 `游戏助手` |
