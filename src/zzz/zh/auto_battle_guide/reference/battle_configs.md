战斗配置存放在 `config/auto_battle/` 目录下，均以 `.sample.yml` 结尾。

在一条龙的**自动战斗**设置页面中选择要使用的战斗配置。

> **注意**：不要修改 `.sample.yml` 文件，代码同步时会被覆盖。如需自定义，请复制并重命名后再编辑。

---

## 配置列表

| 配置名称 | 说明 |
|---|---|
| `全配队通用` | 通用配置，根据角色类型自动分派站场/速切/连携等策略。适用于大多数配队。详见 [架构解析](../advanced/architecture.md) |
| `专属配队-艾莲` | 艾莲专属配队优化 |
| `专属配队-莱卡恩` | 莱卡恩专属配队优化 |
| `专属配队-莱特` | 莱特专属配队优化 |
| `击破站场-强攻速切` | 击破角色站场 + 强攻角色速切 |
| `异常站场-强攻速切` | 异常角色站场 + 强攻角色速切 |
| `强攻站场-击破支援速切` | 强攻角色站场 + 击破/支援角色速切 |
| `自动守护` | 防护角色为主的自动守护模式 |

## 如何选择

- **不知道选什么** → 使用 `全配队通用`，它能自动适配大多数配队
- **有特定角色** → 如果有对应的专属配队，优先使用专属配队
- **想自定义** → 复制一个现有配置，根据 [基础教程](../basic/basic_01_intro.md) 进行修改
