## 首次使用检查清单

按照以下顺序逐项确认，全部通过后即可开始使用一条龙。

不熟悉左侧各入口用途时，可先查看[界面导航](./gui_guide.md)。

### 1. 安装与启动

- [ ] 已下载最新版安装包（推荐 `-Full-Environment` 超完整包）
- [ ] 解压到**固态硬盘**的**纯英文、无空格**路径（如 `D:\ZZZ-OD`）
- [ ] 以管理员身份运行安装器完成安装
- [ ] 使用 `OneDragon-Launcher.exe` 能正常启动 GUI

安装细节请参考[快速开始](./quickstart.md)。

### 2. 游戏内设置（必须）

打开游戏，进入设置，逐项确认：

1. - [x] `「设置」-「画面」-「显示模式」` 设为 **1920x1080 窗口**
        - 游戏分辨率需要是 16:9 比例，如 `1920x1080`，其他分辨率若存在问题请尝试切换
        - 如果使用 `全屏模式` ，则需要保证**显示器分辨率**为 16:9，否则请使用窗口模式
        - 使用过程中请保证游戏图像画面完整，不被任何 `窗口/悬浮加速球/任务栏/FPS显示工具` 所遮挡
2. - [x] `「设置」-「画面」-「帧率」` ~~建议限制在 `30帧`~~，正常配置电脑建议限制在 `60帧`
        - 请不要开启 `无限帧率`，过低帧数、画面卡顿或截图频率不足都可能导致识别出错
        - 画质不需要盲目拉高，以游戏画面稳定、脚本截图流畅为准
3. - [x] 请<font color="red">不要开启</font>任何可能改变显示器颜色的驱动或软件，包括但不限于
        - HDR（通过一条龙启动会自动禁用游戏HDR）
        - 护眼模式/夜间模式
        - N 卡滤镜（快捷键为 ALT+Z ）
        - XBOX 滤镜（快捷键为 WIN+G ）
4. - [x] `「设置」-「输入」-「镜头灵敏度」` 设为 **2 或 3** 可以保证脚本转视角时不会幅度过大错过识别点位；首次使用、切换鼠标/手柄或发现转向异常时，请运行 `「一条龙」-「灵敏度校准」`
5. - [x] `「设置」-「输入」-「镜头自动跟随转动」` 设为 **关闭**，功能[拿命验收](./feat/feat_one_dragon/life_on_line.md)模拟按键以此为基础编写
6. - [x] `「设置」-「其他」-「战斗伤害跳字显示」` 设为 **关闭** 或 **伤害跳字整合** 可以避免部分情况下伤害数字遮挡画面
7. - [x] `「设置」-「其他」-「字体选项」` 选择 **全局细体** 可以有效避免文本识别不准的问题

::: important 警告
游戏内默认预设的「显示模式」、「镜头自动跟随转动」、「战斗伤害跳字显示」和「字体选项」，不太符合一条龙运行要求，请务必按照本页面进行设置，否则可能导致脚本无法正常运行，如识别失败、任务无法完成等问题。
:::

### 3. 脚本设置（必须）

在一条龙 GUI 中逐项配置：

1. - [x] `「账户管理」-「多账户管理」-「账户列表」` 如果你拥有多个账号，请在这里添加不同的一条龙实例，不同实例之间设置会区分开需要分别填写
2. - [x] `「账户管理」-「多账户管理」-「当前账户设置」-「游戏路径」` 指向游戏本体（`games\ZenlessZoneZero Game\ZenlessZoneZero.exe`），而非启动器
3. - [x] `「账户管理」-「多账户管理」-「当前账户设置」-「游戏区服」` 选择你的区服，切换过区服需要重启一条龙
4. - [x] `「代码同步」` 已执行一次更新（或确认已是最新版本）
5. - [x] `「设置」-「游戏设置」-「输入方式」` 推荐选 **剪贴板**，避免输入法冲突
6. - [x] `「设置」-「游戏设置」-「按键设置」` 与游戏内实际按键一致（如果没改过游戏默认键位则不用动）
7. - [x] `「设置」-「资源下载」` OCR 识别和需要用到的识别模型已下载（数字为日期，越新越好）

### 4. 预备编队

- [ ] 在游戏内设置好预备编队
- [ ] 在「一条龙 → 预备编队」点击「开始」自动识别
- [ ] 确认识别结果与游戏内编队名称一致
- [ ] 编队名称建议用中文，避免纯数字（OCR 容易识别错）

### 5. 一条龙运行

- [ ] 按需配置好一条龙运行的功能
- [ ] 回到「仪表盘」首页，确认就绪状态显示正常
- [ ] 点击「启动一条龙」或切换到「一条龙 → 一条龙运行」点击开始
- [ ] 观察右侧日志区域，确认脚本正常运行

::: tip
首次运行建议先不勾选全部功能，只选 2~3 个简单的（如邮件、活跃度奖励）测试一下，确认流程通畅后再全量运行。
:::

## 启动参数

《绝区零》采用 Unity 游戏引擎开发，高级参数见[Unity 文档](https://docs.unity3d.com/Manual/PlayerCommandLineArguments.html)。

启用后，脚本在启动游戏时可以控制游戏的分辨率和窗口，避免自己玩游戏后忘记调整设置导致脚本无法运行。

## 设置页面说明

以下只说明各设置页面的用途，具体配置项见对应页面。

### 游戏设置

在[「设置」→「游戏设置」](./setting/setting_game.md)页面，可以配置游戏基础、后台模式、启动参数和按键设置等内容。

### Overlay

在[「设置」→「Overlay」](./setting/setting_overlay.md)页面，可以配置运行时叠加层、识别绘制、状态面板、性能指标和 patched 截图等内容。

### 资源下载

在[「设置」→「资源下载」](./setting/setting_resource_download.md)页面，可以更新启动器、OCR 识别和专用识别模型。

### 脚本环境

在[「设置」→「脚本环境」](./setting/setting_env.md)页面，可以配置截图方法、调试模式、代码源、Python/Pip 下载源、网络代理和脚本按键等内容。

### 通知设置

在[「设置」→「通知设置」](./setting/setting_notify.md)页面，可以配置运行结果的通知推送。

### 自定义设置

在[「设置」→「自定义设置」](./setting/setting_custom.md)页面，可以配置界面语言、界面主题、自定义主题色和主页背景等外观选项。
