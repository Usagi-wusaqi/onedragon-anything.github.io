## 整体布局

打开一条龙后，你会看到左侧是导航栏，右侧是对应的内容区域。以下从上到下介绍每个导航项。

## 仪表盘（首页）

进入脚本后看到的第一个页面，展示公告信息和快捷操作。

- 公告卡片：显示项目最新动态
- 就绪状态：显示当前配置是否齐全（需要游戏路径、识别模型等都配置好）
- 启动按钮：一键启动一条龙

## 游戏助手

包含战斗助手和委托助手两个子页面。详细说明见[游戏助手功能文档](./feat_game_assistant.md)。

## 一条龙

日常全清的核心页面，包含以下子标签：

| 子页面 | 用途 |
| --- | --- |
| 一条龙运行 | 勾选要跑的日常功能，点击开始一键全清 |
| [预备编队](./feat_one_dragon/onedragon.md#预备编队) | 配置和识别游戏内的预备编队 |
| [体力计划](./feat_one_dragon/charge_plan.md) | 配置体力刷本的计划 |
| [灵敏度校准](./feat_one_dragon/onedragon.md#灵敏度校准) | 校准鼠标/手柄的转向灵敏度 |

此外，在「一条龙运行」子页面中，每个功能卡片旁有 ⚙️ 图标，点击可进入该功能的专属配置（如恶名狩猎计划、咖啡计划、式舆防卫战等）。

## 应用运行

自由选择单个功能运行。详见[应用运行功能文档](./feat_standalone_app.md)。

## 画中画

导航栏中间有一个画中画开关按钮（播放图标）。

开启后，当你把游戏窗口切到后台（比如切回桌面或其他窗口）时，脚本会自动弹出一个小窗口实时显示游戏画面。切回游戏后画中画自动隐藏。

行为说明：
- 点击画中画小窗口 → 游戏窗口自动切到前台
- 右键关闭画中画 → 暂停截图，下次游戏切到前台后重置
- 画中画会记住上次的位置和大小
- 开启状态在重启脚本后会自动恢复

## 底部导航

从上到下依次是：

| 按钮 | 用途 |
| --- | --- |
| 点赞 | 前往项目赞助/支持页面 |
| 开发工具 | 开发者使用的截图助手、指令调试、代理人模板生成等工具 |
| 代码同步 | 检测并更新脚本代码版本，也可查看当前版本哈希 |
| 账户管理 | 添加多个游戏账号实例，每个实例的配置互相独立 |
| 设置 | 游戏设置、资源下载、脚本环境、通知设置、自定义设置 |

## 设置

点击「设置」进入后，有以下子页面：

| 子页面 | 用途 |
| --- | --- |
| 游戏设置 | 输入方式、窗口尺寸、操控方式、后台模式、启动参数、按键设置 |
| 资源下载 | 下载和更新识别模型（闪光识别、空洞格子、迷失之地） |
| 脚本环境 | 代理设置、Python 环境、OCR 缓存 |
| 通知设置 | 运行结果推送（Server酱、Bark、钉钉等） |
| 自定义设置 | 代理地址等高级选项 |

各页面的详细配置说明见[必要设置](./config.md#设置页面说明)。
