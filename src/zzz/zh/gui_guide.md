## 整体布局

打开一条龙后，你会看到左侧是导航栏，右侧是对应的内容区域。以下从上到下介绍每个导航项。

> 以脚本标签页为单位分类介绍

| 导航项 | 说明 |
| --- | --- |
| [仪表盘](./feat/feat_dashboard.md) | 查看公告、检查资源更新，并提供启动一条龙入口 |
| [游戏助手](./feat/feat_game_assistant.md) | 集成战斗助手和委托助手，包括自动战斗、闪避助手、自动对话/剧情等功能 |
| [一条龙](#一条龙) | 管理一条龙日常流程及相关配置，包括一条龙运行、体力计划、预备编队和灵敏度校准 |
| [应用运行](./feat/feat_standalone_app.md) | 添加并单独运行一条龙中的应用任务，适合临时执行单个功能 |
| [画中画](./feat/pip.md) | 游戏切到后台时显示置顶预览窗口，点击窗口可快速切回游戏 |
| 点赞 | 前往 GitHub Star、访问 GitHub 指南、赞赏及查看贡献者信息 |
| 开发工具 | 提供图像分析、模板管理、画面管理、代理人模板生成、截图助手、指令调试等开发和排障工具 |
| 代码同步 | 配置自动更新、强制更新、自定义分支，并手动同步代码 |
| 账户管理 | 管理多账号实例、游戏路径、区服和账号信息 |
| [设置](#设置) | 配置游戏设置、Overlay、资源下载、脚本环境、通知设置和自定义设置 |

## 一条龙

| 子页面 | 用途 |
| --- | --- |
| [一条龙运行](./feat/feat_one_dragon/onedragon.md#一条龙运行) | 勾选要跑的日常功能，点击开始一键全清 |
| [体力计划](./feat/feat_one_dragon/charge_plan.md) | 配置体力刷本的计划 |
| [预备编队](./feat/feat_one_dragon/onedragon.md#预备编队) | 配置和识别游戏内的预备编队 |
| [灵敏度校准](./feat/feat_one_dragon/onedragon.md#灵敏度校准) | 校准鼠标/手柄的转向灵敏度 |

## 设置

| 子页面 | 用途 |
| --- | --- |
| [游戏设置](./setting/setting_game.md) | 配置输入方式、后台模式、启动参数、HDR 切换和按键设置等 |
| [Overlay](./setting/setting_overlay.md) | 配置运行时叠加层、识别绘制、状态面板、性能指标和 patched 截图 |
| [资源下载](./setting/setting_resource_download.md) | 更新启动器、OCR 识别和专用识别模型 |
| [脚本环境](./setting/setting_env.md) | 配置截图方法、调试模式、代码源、Python/Pip 下载源、网络代理和脚本按键等 |
| [通知设置](./setting/setting_notify.md) | 配置运行结果推送、推送渠道、代理和测试通知 |
| [自定义设置](./setting/setting_custom.md) | 配置界面语言、主题、主题色和主页背景等外观选项 |
