## 整体布局

打开一条龙后，你会看到左侧是导航栏，右侧是对应的内容区域。以下从上到下介绍每个导航项。

> 以脚本标签页为单位分类介绍

| 导航项 | 说明 |
| --- | --- |
| [仪表盘](./feat/feat_dashboard.md) | 查看公告、检查资源更新，并提供启动一条龙入口 |
| [游戏助手](#游戏助手) | 集成战斗助手和委托助手，包括自动战斗、闪避助手、自动对话/剧情、钓鱼等功能 |
| [一条龙](#一条龙) | 集成大量日常功能，如体力刷本、零号空洞、锄大地 |
| [应用运行](#应用运行) | 添加并单独运行一条龙中的应用任务，适合临时执行单个功能 |
| [画中画](./feat/pip.md) | 游戏切到后台时显示置顶预览窗口，点击窗口可快速切回游戏 |
| [点赞](#底部导航) | 前往 GitHub Star、访问 GitHub 指南、赞赏及查看贡献者信息 |
| [开发工具](#底部导航) | 提供图像分析、模板管理、画面管理、截图助手、指令调试等开发和排障工具 |
| [代码同步](#底部导航) | 配置自动更新、强制更新、自定义分支，并手动同步代码 |
| [账户管理](#底部导航) | 管理多账号实例、游戏路径、区服和账号信息 |
| [设置](#设置页面) | 配置游戏设置、资源下载、脚本环境、通知设置和自定义设置 |

## 仪表盘（首页）

进入脚本后看到的第一个页面，用于查看公告、快速打开常用链接、检查更新状态，并作为一条龙的快捷启动入口。详见[仪表盘说明](./feat/feat_dashboard.md)。

## 游戏助手

包含战斗助手和委托助手两个子页面。详细说明见[游戏助手功能文档](./feat/feat_game_assistant.md)。

## 一条龙

日常全清的核心页面，包含以下子标签：

| 子页面 | 用途 |
| --- | --- |
| [一条龙运行](./feat/feat_one_dragon/onedragon.md#一条龙运行) | 勾选要跑的日常功能，点击开始一键全清 |
| [预备编队](./feat/feat_one_dragon/onedragon.md#预备编队) | 配置和识别游戏内的预备编队 |
| [体力计划](./feat/feat_one_dragon/charge_plan.md) | 配置体力刷本的计划 |
| [灵敏度校准](./feat/feat_one_dragon/onedragon.md#灵敏度校准) | 校准鼠标/手柄的转向灵敏度 |

此外，在「一条龙运行」子页面中，每个功能卡片旁有 ⚙️ 图标，点击可进入该功能的专属配置（如恶名狩猎计划、咖啡计划、式舆防卫战等）。

## 应用运行

自由选择单个功能运行。详见[应用运行功能文档](./feat/feat_standalone_app.md)。

## 画中画

导航栏中间有一个画中画开关按钮（播放图标）。

开启后，当你把游戏窗口切到后台（比如切回桌面或其他窗口）时，脚本会自动弹出一个小窗口实时显示游戏画面。切回游戏后画中画自动隐藏。详见[画中画功能文档](./feat/pip.md)。

## 底部导航

从上到下依次是：

| 按钮 | 用途 |
| --- | --- |
| 点赞 | 前往项目赞助/支持页面 |
| 开发工具 | 开发者使用的截图助手、指令调试、代理人模板生成等工具 |
| 代码同步 | 检测并更新脚本代码版本，也可查看当前版本哈希 |
| 账户管理 | 添加多个游戏账号实例，每个实例的配置互相独立 |
| 设置 | 游戏设置、资源下载、脚本环境、通知设置、自定义设置 |

## 设置页面

点击「设置」进入后，有以下子页面。已有单独说明的页面以对应文档为准，本页只说明入口用途。

| 子页面 | 用途 | 详细说明 |
| --- | --- | --- |
| 游戏设置 | 配置输入方式、后台模式、启动参数、按键设置等 | [必要设置-游戏设置](./config.md#游戏设置) |
| 资源下载 | 更新识别模型和启动器资源 | [必要设置-资源下载](./config.md#资源下载) |
| 脚本环境 | 配置截图方法、代码源、Python/Pip 下载源、网络代理和脚本按键等 | [必要设置-脚本环境](./config.md#脚本环境) |
| 通知设置 | 配置运行结果推送 | [必要设置-通知设置](./config.md#通知设置) |
| 自定义设置 | 配置界面语言、主题和主页背景等外观选项 | [必要设置-自定义设置](./config.md#自定义设置) |
