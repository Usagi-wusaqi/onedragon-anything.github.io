本页对应一条龙界面左侧导航 `「设置」-「Overlay」` 中的配置项，主要说明运行时叠加层、识别绘制、状态面板、性能指标和 patched 截图。不熟悉左侧各入口用途时，可先查看[界面导航](../gui_guide.md#设置页面)。

::: tip
Overlay 是调试和观察运行状态用的叠加层。普通日常运行不需要开启。
:::

## Overlay 基础

### 启用 Overlay

开启后，可以通过 `Ctrl+Alt+O` 切换 Overlay 显隐。主键可以在 `显隐热键主键` 中修改。

### 显隐热键主键

组合键固定为 `Ctrl+Alt`，这里只配置最后一个主键。

### 默认显示

控制启动后 Overlay 是否默认可见。

### 防截图保护

开启后使用 `WDA_EXCLUDEFROMCAPTURE` 隐藏 Overlay，避免它被系统截图捕获。

该能力依赖 Windows 10 2004 及以上版本；系统版本过低时，Overlay 会被禁用。

## 视觉绘制

`视觉绘制` 用于把识别结果叠加到游戏画面上，方便排查识别问题。

可分别控制：

- `启用视觉层`
- `显示 YOLO`
- `显示 OCR`
- `显示 Template`
- `显示 CV Pipeline`

如果识别框和实际画面有偏移，可以调整：

- `视觉层 X 偏移`
- `视觉层 Y 偏移`
- `视觉层 X 缩放`
- `视觉层 Y 缩放`

## 面板与刷新

Overlay 可显示多种运行状态面板：

- `显示日志面板`
- `显示状态面板`
- `显示决策链路面板`
- `显示时间轴面板`
- `显示性能面板`

开启 `面板编辑模式` 后，可以拖拽调整面板位置，并可调整日志窗字体和透明度。

相关配置：

- `面板字体大小`
- `文字颜色`
- `日志最大行数`
- `日志过期秒数`
- `窗口跟随间隔(ms)`
- `状态刷新间隔(ms)`
- `面板透明度(%)`
- `重置面板位置`

## 性能指标

性能面板中可以单独开关以下指标：

- `OCR 耗时`
- `YOLO 耗时`
- `CV Pipeline 耗时`
- `节点轮次耗时`
- `Overlay 刷新耗时`

## 截图

### 保存 patched 合成图

开启后，脚本保留原截图和剪贴板行为，同时额外输出一张叠加 Overlay 的 patched 图片。

### patched 文件后缀

设置 patched 图片的文件后缀，默认是 `_patched`。
