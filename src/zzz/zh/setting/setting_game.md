本页对应一条龙界面左侧导航 `「设置」-「游戏设置」` 中的配置项，主要说明游戏基础、后台模式、启动参数和按键设置等内容。不熟悉左侧各入口用途时，可先查看[界面导航](../gui_guide.md#设置页面)。

::: important
`后台模式（测试版）`、`操控方式` 中的 `Xbox` / `DS4`、以及 `手柄按键` 都依赖虚拟手柄。

脚本会通过 `vgamepad` + `ViGEmBus` 在系统中创建一个额外的虚拟手柄设备，由游戏像普通手柄一样读取它的输入。未检测到依赖时，相关功能会提示不可用，并切回不依赖手柄的设置。

安装方式：

1. 安装器：在安装器的 `虚拟手柄` 卡片中安装依赖；如果弹出驱动安装窗口，按正常流程允许安装。
2. 源码环境：在项目根目录执行：

   ```shell
   uv sync --group gamepad
   ```

[ViGEmBus 官方发布页](https://github.com/nefarius/ViGEmBus/releases/latest)
:::

## 1.游戏基础

### 1.1.输入方式

`输入方式` 用于脚本需要输入文本时选择输入来源。

- `剪贴板`：通过剪贴板写入文本，通常是默认选择。
- `键盘输入`：模拟键盘逐字输入，使用时需要确保没有启用输入法。

如果出现剪贴板写入失败，可以切换到 `键盘输入` 尝试。

### 1.2.后台模式（测试版）

`后台模式（测试版）` 用于在游戏窗口不置顶时执行部分操作。它需要虚拟手柄驱动和 `PrintWindow` 截图方式，运行时会短暂抢占鼠标进行点击操作。

相关配置：

- `后台模式（测试版）`：开启或关闭后台模式。
- `后台手柄类型`：选择后台模式使用 `Xbox` 还是 `DS4`。
- `闪切时长（秒）`：后台模式切换键鼠输入时的前台停留时长，过小可能切换失败。
- `菜单` / `地图` / `小地图` / `快捷手册` / `功能导览`：配置后台模式下用手柄按键替代点击的动作键。

注意：

- 后台模式依赖虚拟手柄。未安装时会提示 `未检测到 vgamepad / ViGEmBus`，并自动关闭。
- 不建议在前台同时玩其它支持手柄的游戏，避免虚拟手柄输入被其它游戏接收。

### 1.3.切换 HDR 状态

`切换 HDR 状态` 会修改当前游戏路径在 Windows 注册表中的 Auto HDR 设置。

- `禁用 HDR`：将当前游戏路径设置为禁用 Auto HDR。
- `启用 HDR`：将当前游戏路径设置为启用 Auto HDR。

这个设置仅影响手动启动游戏。一条龙启动游戏时会自动禁用 HDR，用来避免 HDR 改变画面颜色后影响识别。

### 1.4.启动参数

`启动参数` 控制一条龙启动游戏时附加的窗口参数。开启后，可以统一游戏窗口尺寸和显示方式，避免手动游玩后忘记恢复设置。

相关配置：

- `窗口尺寸`：可选 `1920x1080` / `2560x1440` / `3840x2160`。
- `全屏`：可选 `窗口化` 或 `全屏`。
- `无边框窗口`：开启后使用无边框窗口。
- `显示器序号`：选择启动到第几个显示器。
- `高级参数`：额外启动参数，不清楚用途时不要填写。

## 2.按键设置

### 2.1.操控方式

`操控方式` 仅影响自动战斗 / 闪避助手运行时使用的控制方式。

- `键鼠`：使用键盘和鼠标输入。
- `Xbox`：使用 Xbox 虚拟手柄输入。
- `DS4`：使用 DS4 虚拟手柄输入。

选择 `Xbox` 或 `DS4` 前依赖虚拟手柄。未检测到依赖时，会提示 `手柄操控不可用` 并切回 `键鼠`。

### 2.2.键盘按键

`键盘按键` 用于配置键鼠模式下的动作映射。如果游戏内改过键位，需要在这里同步修改，否则自动操作会按错键。

这里包含 `交互`、`普通攻击`、`闪避`、`角色切换`、`特殊攻击`、`终结技`、`连携技`、`移动`、`锁定敌人` 等动作。

### 2.3.手柄按键

`手柄按键` 用于配置 `Xbox` 和 `DS4` 两套虚拟手柄映射。

相关配置：

- 顶部下拉框：切换当前正在配置的手柄类型。
- `单次按键持续时间(秒)`：按键按下后保持的时间，过小可能按键被吞，过大可能影响操作节奏。
- 各动作按键：配置 `交互`、`普通攻击`、`闪避`、`角色切换`、`特殊攻击`、`终结技`、`连携技`、`移动`、`锁定敌人` 等动作对应的手柄按键。
