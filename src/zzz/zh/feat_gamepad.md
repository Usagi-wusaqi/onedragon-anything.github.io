::: important
当前版本战斗助手的手柄相关逻辑似乎存在 bug。若遇到 `操作方式` 设置后不生效，或 `自动战斗` / `闪避助手` 表现异常，请同时检查 `战斗助手 -> 操作方式` 和 `设置 -> 游戏设置 -> 按键设置` 中的相关配置。
:::

手柄功能通过虚拟手柄实现：脚本通过 `vgamepad` + `ViGEmBus` 在系统中创建一个额外的虚拟手柄设备，由游戏像普通手柄一样读取它的输入。

目前支持 `Xbox` 和 `DS4` 两种类型，主要用于以下场景：

- 战斗助手 / 闪避助手 的战斗操作
- 后台模式（测试版）下，替代部分锁鼠标场景中的点击和移动操作

## 1.前置依赖

手柄功能依赖 `vgamepad` 和 `ViGEmBus` 驱动。未安装时，战斗助手里的手柄选项和后台模式都会提示不可用。

### 1.1.安装器

在安装器的 `虚拟手柄` 卡片中安装即可。

安装过程中如果弹出驱动安装窗口，接受安装即可。

[ViGEmBus 官方发布页](https://github.com/nefarius/ViGEmBus/releases/latest)

### 1.2.手动安装

源码环境可在项目根目录执行：

```shell
uv sync --group gamepad
```

如果你不是用安装器，而是自己维护 Python 环境，至少也要保证 `vgamepad` 可用，并完成 `ViGEmBus` 驱动安装。

## 2.按键设置

如果游戏中改过键位，需要在脚本中同步修改，否则自动操作会按错键。

目录为 `设置` -> `游戏设置` -> `按键设置`

这里包含：

- `键盘 / Xbox / DS4` 三套按键映射
- `手柄按键` 中分别配置 `Xbox` 和 `DS4` 的具体按键
- `单次按键持续时间(秒)`，需按机器情况微调，过小可能按键被吞，过大可能影响操作

## 3.战斗助手里怎么用

战斗助手页面的 `操作方式` 才是自动战斗 / 闪避助手运行时实际使用的控制方式。

目录为 `战斗助手` -> `操作方式`

说明：

- 选 `键鼠` 时，不使用虚拟手柄
- 选 `Xbox` / `DS4` 时，启动前会检查 `vgamepad`
- 实际按键映射仍然读取 `设置` -> `游戏设置` -> `按键设置` 中的配置

## 4.后台模式（测试版）

现在手柄不只用于战斗助手，也用于后台模式。

目录为 `设置` -> `游戏设置` -> `游戏基础` -> `后台模式（测试版）`

相关配置：

- `后台模式（测试版）`：开启后可在游戏窗口不置顶时执行部分操作
- `后台手柄类型`：选择后台模式使用 `Xbox` 还是 `DS4`
- `闪切时长（秒）`：后台模式切换键鼠输入时的停留时长
- 后台模式下的 `手柄动作键`：可给 `菜单`、`地图`、`小地图`、`快捷手册`、`功能导览` 配置对应手柄键

注意：

- 后台模式要求虚拟手柄依赖，并且截图方式需要使用 `PrintWindow`
- 运行时会短暂抢占鼠标进行点击操作，无需游戏窗口置顶
- 不建议在前台同时操作其它支持手柄的游戏
