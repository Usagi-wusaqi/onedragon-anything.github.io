## 首次运行检查清单

按照以下顺序逐项检查，全部通过后即可开始使用一条龙。

### 1. 安装 ✅

- [ ] 已下载最新版安装包（推荐 `-Full-Environment` 超完整包）
- [ ] 解压到**固态硬盘**的**纯英文、无空格**路径（如 `D:\ZZZ-OD`）
- [ ] 以管理员身份运行安装器完成安装
- [ ] 使用 `OneDragon-Launcher.exe` 能正常启动 GUI

::: details 安装细节
详细安装步骤请参考[快速开始](./quickstart.md)。

路径要求：
- ❌ 不要用含中文的路径（如 `C:\用户\...`）
- ❌ 不要用含空格的路径
- ❌ 不要放在机械硬盘
- ❌ 路径不要太长
:::

### 2. 游戏内设置 ✅

打开游戏，进入设置，逐项确认：

- [ ] 「设置 → 画面 → 显示模式」设为 **1920×1080 窗口**
- [ ] 「设置 → 画面 → 帧率」设为 **60 帧**或更高
- [ ] 未开启 HDR、护眼模式、N卡滤镜、Xbox 滤镜等可能改变画面颜色的功能
- [ ] 「设置 → 输入 → 镜头灵敏度」设为 **2 或 3**
- [ ] 「设置 → 输入 → 镜头自动跟随转动」设为 **关闭**
- [ ] 「设置 → 其他 → 战斗伤害跳字显示」设为 **关闭** 或 **伤害跳字整合**
- [ ] 「设置 → 其他 → 字体选项」设为 **全局细体**

::: warning
以上设置未按要求配置会导致脚本识别失败或运行异常。
:::

### 3. 脚本设置 ✅

在一条龙 GUI 中逐项配置：

- [ ] 「账户管理 → 多账户管理 → 游戏路径」指向游戏本体（`ZenlessZoneZero.exe`，不是启动器）
- [ ] 「账户管理 → 多账户管理 → 游戏区服」选择你的区服
- [ ] 「代码同步」已执行一次更新（或确认已是最新版本）
- [ ] 「设置 → 游戏设置 → 输入方式」推荐选 **剪贴板**
- [ ] 「设置 → 游戏设置 → 按键设置」与游戏内实际按键一致（如果没改过游戏默认键位则不用动）
- [ ] 「设置 → 资源下载」三个模型都已下载（数字为日期，越新越好）

### 4. 预备编队 ✅

- [ ] 在游戏内设置好预备编队
- [ ] 在「一条龙 → 预备编队」点击「开始」自动识别
- [ ] 确认识别结果与游戏内编队名称一致
- [ ] 编队名称建议用中文，避免纯数字（OCR 容易识别错）

### 5. 日常功能配置（按需）✅

- [ ] [体力计划](./feat_one_dragon/charge_plan.md)：添加要刷的副本和次数
- [ ] [恶名狩猎计划](./feat_one_dragon/notorious_hunt.md)：配置每周要打的 Boss
- [ ] [式舆防卫战](./feat_one_dragon/shiyu_defense.md)：配置弱点配队
- [ ] [咖啡计划](./feat_one_dragon/coffee.md)：选择咖啡类型和喝后行为

以上不配置也能运行一条龙，但对应功能会被跳过或使用默认配置。

### 6. 启动！🚀

- [ ] 回到「仪表盘」首页，确认就绪状态显示 ✅
- [ ] 点击「启动一条龙」或切换到「一条龙 → 一条龙运行」点击开始
- [ ] 观察右侧日志区域，确认脚本正常运行

::: tip
首次运行建议先不勾选全部功能，只选 2~3 个简单的（如邮件、活跃度奖励）测试一下，确认流程通畅后再全量运行。
:::

### 遇到问题？

1. 查看[常见问题](./faq.md)
2. 检查日志文件（游戏根目录下 `.log/log.txt`）
3. 按 F11 可截取游戏画面截图，保存在 `.debug/images` 文件夹
4. 到 [Issues](https://github.com/OneDragon-Anything/ZenlessZoneZero-OneDragon/issues) 搜索或提问
